# 🧠 Jogo da Memória

Bem-vindo(a) ao **Jogo da Memória**!

Este projeto foi feito com **HTML**, **CSS** e **JavaScript**!  
Um jogo simples, divertido e desafiador para testar sua memória combinando pares de cartas com imagens fofas e coloridas. 🎉

---

## ✨ Funcionalidades

- ✅ Geração dinâmica de cartas
- 🔄 Cartas embaralhadas a cada nova execução
- 👆 Clique para virar e combinar os pares
- 💥 Animação suave de virada
- 📱 Totalmente responsivo (funciona em celular e tablet)

---

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3 (com media queries para responsividade)
- JavaScript (manipulação de DOM, lógica e interação)

---

## 📁 Estrutura do Projeto

📦 Memory-Game/
┣ 📁 `assets/`    -> onde estão as imagens das cartas (ex: img1.png, img2.png ...)
┣ 📜 `index.html` → Estrutura da página
┣ 📜 `style.css`  → Estilos visuais e responsividade
